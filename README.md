# 双无人机协同避障 - Q-learning实现

## 项目简介

本项目使用Q-learning强化学习算法，在10x10栅格地图中训练两个无人机智能体，使它们从各自起点出发，避开障碍物并避免相互碰撞，最终到达各自目标点。

## 环境要求

- Python 3.8+
- numpy
- matplotlib

安装依赖：

pip install numpy matplotlib

## 文件结构

无人机项目/
├── environment.py      # 环境定义（地图、障碍物、起点终点）
├── agent.py            # Q-learning智能体
├── train.py            # 训练主程序
├── test.py             # 测试和可视化
├── q_table_a.npy       # 无人机A的Q表（训练后生成）
├── q_table_b.npy       # 无人机B的Q表（训练后生成）
├── training_curves.png # 训练曲线图
├── path_visualization.png # 路径可视化图
└── README.md           # 项目说明

## 运行方法

### 1. 训练模型

python train.py

训练500轮，约需2-5分钟。训练过程中会打印进度：

轮数：150/500 | 成功率：100.0% | 平均奖励：-8.72 | 平均步数：37.2
...
训练完成！

### 2. 测试模型

python test.py

测试会：
- 加载训练好的Q表
- 执行单次测试并显示路径图

- 输出策略分析

## 环境设置

### 地图参数

- 大小：10x10栅格
- 障碍物：
  - 2x2障碍物块 at (2,2)-(3,3)
  - 一列障碍物 at (5,7),(6,7),(7,7)
  - 单个障碍物 at (8,4)

### 起点与终点

| 智能体 | 起点 | 终点 |
|--------|------|------|
| 无人机A | (1,1) | (8,8) |
| 无人机B | (8,9) | (1,1) |

### 动作空间

| 编号 | 动作 |
|------|------|
| 0 | 上 |
| 1 | 下 |
| 2 | 左 |
| 3 | 右 |
| 4 | 停留 |

### 奖励函数

| 情况 | 奖励 |
|------|------|
| 到达目标 | +10 |
| 碰撞 | -5 |
| 每步 | -0.1 |

## 超参数

| 参数 | 符号 | 值 | 说明 |
|------|------|-----|------|
| 学习率 | α | 0.1 | 控制学习速度 |
| 折扣因子 | γ | 0.9 | 重视未来奖励 |
| 探索率 | ε | 0.2 | 随机探索概率 |

## 常见问题

Q1: 中文显示为方框？

解决：在代码开头添加
plt.rcParams['font.sans-serif'] = ['SimHei', 'Microsoft YaHei']
plt.rcParams['axes.unicode_minus'] = False

Q2: 提示未找到Q表文件？

解决：先运行train.py训练模型，再运行test.py

Q3: 每次训练路径不同？

正常现象：Q-learning存在随机探索，且存在多条等长最优路径。

## 算法原理

Q-learning更新公式：

Q(s,a) ← Q(s,a) + α × [r + γ × maxQ(s') - Q(s,a)]

- s：当前状态（位置）
- a：执行的动作
- r：获得的奖励
- s'：下一个状态
